Скрипты для резервного копирования файлов Вашего VDS/VPS сервера на Яндекс.Диск

Скрипты содержат опции для автоматического удаления старых резервных копий.

На сервере после завершения работы скриптов остаётся только по одной копии каждого вида
(или несколько, если интервал между их созданием менее 30 минут)

Все каталоги и подкаталоги, указнные в переменных, должны быть созданы заранее!

Яндекс.Диск должен быть создан и примонтирован в системе. Точка монтирования - /mnt/yadisk

При использовании всех трёх скриптов (ежедневного, еженедельного и ежемесячного копирования, с выполнением по cron'у) должны получаться следующие копии:

1. Ежедневные копии за последние 6 дней
2. Еженедельные копии за последние 30 дней
3. Ежемесячные копии за последние 366 дней

Все резервные копии формируется в отдельных каталогах.
